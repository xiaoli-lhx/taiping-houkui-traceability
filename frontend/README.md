# Frontend

本目录为太平猴魁茶叶溯源与品质辅助评估系统的前端项目，基于 `React + TypeScript + Vite + Ant Design` 构建。

## Features

- 登录、注册与角色化入口
- 后台管理页面
- 茶叶批次与溯源信息展示
- 品质评估结果展示
- 消费者公开查询与个人功能页
- 数据统计图表展示

## Development

在 `frontend` 目录下执行：

```bash
npm install
npm run dev
```

默认开发地址：

- `http://127.0.0.1:3000`

## Build

```bash
npm run build
```

构建产物输出到 `frontend/dist`。
