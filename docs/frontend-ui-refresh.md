# 前端 UI 重构说明

## 1. 重构目标

本次前端重构不调整后端接口，只将原有简易自定义页面升级为更适合毕业答辩展示的后台版本。

核心目标：

- 使用 `Ant Design` 统一后台页面风格
- 使用 `@ant-design/plots` 展示图表
- 在消费者查询页提供真实二维码
- 强化角色菜单和页面权限体验
- 支持管理员门户、注册页与消费者中心

## 2. 当前前端栈

- React
- Vite
- Ant Design
- Ant Design Charts
- qrcode.react

## 3. 主要页面变化

### 登录页

- 角色选择 + 用户名 + 密码
- 正常系统登录样式
- 提供匿名公开查询入口和注册页入口

### 注册页

- 茶农、企业、消费者可自助注册
- 注册后进入待审核状态

### 管理员门户

- 注册申请审核
- 用户管理

### 批次管理页

- 批次表格展示
- 企业角色使用 Drawer 新建批次
- 茶农角色不显示批次创建入口

### 批次详情页

- 基础信息使用 Descriptions 展示
- 阶段链路使用 Timeline 展示
- 阶段新增改为 Drawer 表单
- 审核记录使用 Table 展示

### 品质评估页

- 表单录入指标
- 统计卡片展示得分和等级
- 雷达图展示品质结果

### 统计分析页

- 概览指标卡
- 产量分布柱状图
- 品质等级环图
- 品质指标趋势折线图

### 监管审核页

- 审核批次列表
- 批次审核详情
- Drawer 提交审核记录

### 公开查询页 / 消费者中心

- 支持 `?code=` 参数自动查询
- 真实二维码
- 品质雷达图
- 阶段链路时间线
- 支持通过 `VITE_PUBLIC_QUERY_BASE_URL` 配置二维码中的局域网或公网访问地址
- 消费者登录后会记录查询历史，并支持收藏与反馈
